# File Sharing Application

A secure file sharing application built with Node.js and Express.

## Features

- File upload and sharing
- Password protection
- Link expiration
- Download tracking

## Setup

1. Clone the repository
2. Run `npm install`
3. Start the server: `npm start`

## Environment Variables

Create a `.env` file:

```
PORT=3000
HOST=0.0.0.0
CORS_ORIGIN=*
```
